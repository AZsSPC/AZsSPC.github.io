How to use it?

Single command - sNaF:
s - [r or c] row or column
N - row/column number
F - row/column number or number
a - [+ - / *>] the action to be applied
  • + - adds all cells values from F to N, writing in N
  • - - subtracts all cells values of F from N and writes to N
  • * - multiplies each cell N by the value F, writing to N
  • / - divides each cell N by the value F, writing in N
  • > - moves the row/column to the specified index, shifting all others
